BC Fabrication - Easy Edit Version

This version is much easier to maintain.

Edit here:
- content/site.json = all main text, phone, email, headings, SEO
- content/gallery.json = gallery image paths, titles, descriptions
- assets/ = images and logo

Easy editing options:
1. Edit the JSON files directly
2. Open editor.html in a browser, change the content, download new JSON files, replace the old ones, then upload the site to Netlify again

Important:
This keeps the site simple and reliable.
There is no live login or database, so edits do not save automatically online.
The tradeoff is that changes are straightforward and all in one place.
